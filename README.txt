Transform — BUas CMGT Programming Portfolio Game

A puzzle-platformer where you switch between two forms to solve levels.

-Blue (slim): fast, can walk on glass bridges, cannot push heavy blocks
-Red (wide): slow, shatters glass bridges, can push heavy blocks
-Reach the gold goal to win. Collect stars.

Controls:

Move: Arrow keys / WASD 
Jump: Space / W / Up
Switch form: Tab
Restart level: R
Quit: Escape
